# Title # 
Fraudulent Transaction Detection for Digital Money Transfer

# Synopsis #
Novapay is a real-time digital money transfer company. The company is expanding globally and facing significant challenges in maintaining the security of its transactions amidst the increasing sophistication of fraud.

Novapay intends to protect its customers from fraudulent activities such as: 
•	Identity theft 
•	Unauthorized payments, and 
•	Account takeovers 

# Project Goal #
This project aims to develop a machine learning-based fraud detection system for NovaPay, replacing its existing static, rules-based approach. Specifically, Novapay wants the ML-solution to achieve the following:
1. Increase the accuracy of fraud detection
2. Minimise the cases of false positives
3. Be robust, scalable and adaptable in the long-term
4. Meet regulatory standards
5. Be highly efficient

# Target Variables #
The target variables are represented by the binary (1/0) in the data column 'is_fraud'. Each representing;
•	Valid transaction
•	Fraudulent transaction
